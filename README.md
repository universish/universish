# ABOUT

Hello Open Source Friends,

Tired of data mining apps? You've come to the right place. I find and test FLOSS (Free/Libre Open Source Software) alternatives to popular and insecure apps. My goal is to help everyone protect their digital privacy, regardless of their technical knowledge.

What Will You Gain as a Patreon Supporter?

🔓 Early Access: Get review videos and guides before anyone else.

📋 Comprehensive Lists: Curated special lists like “What should I use instead of which app?”

💬 Q&A and Support: Direct communication for setup issues or app recommendations.

🗳️ Content Direction: You decide which open-source project we'll review in the next video.

Thank you for being part of this small but powerful community that values privacy!
